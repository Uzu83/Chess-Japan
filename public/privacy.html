プライバシーポリシー

Chess-Japan — 1手解説AI（https://chess-japan.pages.dev）／最終更新: 2026-07-07

本サービスは個人が運営する無料のチェス学習サイトです。アカウント登録なしで 全機能を利用でき、その場合に氏名・メールアドレス等の個人情報を収集することはありません （任意のログイン機能を使う場合は第4節をご覧ください）。 本ページでは、サービスの動作上取り扱う情報を説明します。

1. ブラウザ内にのみ保存される情報

  • 対局履歴・棋譜・内部レート・解説キャッシュは、お使いのブラウザの localStorage にのみ保存されます。
  • これらが運営者のサーバーへ送信・収集されることはありません。ブラウザのサイトデータ削除でいつでも消去できます。

2. AI 解説リクエスト時に送信される情報

  • 「この一手を解説」等を実行したときのみ、局面情報（FEN）・指し手・エンジン評価値などの 盤面データが当サービスのサーバー（Supabase Edge Function）へ送信され、 解説文生成のため外部の大規模言語モデル API に転送されます。 現在の転送先は Google Gemini API（Google LLC）です。運用上、転送先モデルは変更されることがあります。
  • 盤面データに個人情報は含まれません。生成された解説文は品質・コスト最適化のためサーバー側にキャッシュされます。

3. IP アドレスの取り扱い

  • 解説リクエスト時、濫用防止（レート制限）のため IP アドレスを一時的にサーバーへ保存します。 保存は期限付きで、期限経過後に自動削除されます（最長でも数日以内）。
  • bot 対策として Cloudflare Turnstile（Cloudflare, Inc.）を利用しており、 検証のため IP アドレス等が Cloudflare に送信されます。 詳細は Cloudflare のプライバシーポリシー をご覧ください。
  • ホスティング（Cloudflare Pages）のアクセス処理においても、通信の性質上 IP アドレスが Cloudflare を経由します。

4. アカウント（任意のログイン機能）

  • 本サービスはログインなしで全機能を利用できます。任意で Google アカウントによるログインを利用した場合、認証基盤（Supabase Inc.）に Google アカウントのメールアドレス・表示名が保存され、当サービスは表示名・内部レート・対局数をプロフィールとして保存します。
  • 表示名は初期値として Google の表示名が設定されますが、いつでも変更できます。プロフィールは本人のみが閲覧・編集でき、他のユーザーには公開されません（ランキング等を導入する際は本ページを更新します）。

5. エラー監視（Sentry）

  • サービス品質改善のため、エラー発生時にエラー内容・ブラウザ種別・当サイト内の URL（クエリ等は除去）を Sentry（Functional Software, Inc.）に送信することがあります。
  • 個人を特定する情報を意図的に付与しない設定（sendDefaultPii: false）で運用しています。

6. アクセス解析（導入予定）

  • 今後、Cloudflare Web Analytics（Cookie を使用しない、個人を追跡しない集計型の解析）を 導入する場合があります。導入時は本ページを更新します。

7. お問い合わせ

本ポリシーやサービスに関するお問い合わせは GitHub Issues にお寄せください。 Issues は誰でも閲覧できる公開の場です。個人情報を書き込まないでください。 GitHub 利用時の情報の取り扱いは GitHub, Inc. のプライバシーポリシーに従います。
